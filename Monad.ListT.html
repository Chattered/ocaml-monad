Functor Monad.ListT

module ListT: 
functor (M : BatInterfaces.Monad) -> sig .. end
The list monad transformer will add non-determinism to computations. We have not provided a transformer for lazy lists, since I'm not yet sure how to implement it. It would probably need a lazy version of map_m, but it's not clear to me how to write this, since whether the computations are strict will determine whether the argument has to be completely forced.
