val (>>=) : 'a m -> ('a -> 'b m) -> 'b m
val join : 'a m m -> 'a m
Join unwraps one layer of computation: if you can compute computations, then you have those computations.

For a concrete example, think of lists. Here, values of type 'a list list are computations of non-deterministic computations. Each inner computation returns possible values. If we join the outer computation, then we get all possible values from all inner computations.
