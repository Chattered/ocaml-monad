Module Monad.List

module List: BasePlus  with type 'a m = 'a list
The venerable list monad. Lists can be understood as computations which return zero or more possible values, and where further computation on these values is non-deterministic.
include BatInterfaces.Monad
val zero : unit -> 'a m
val plus : 'a m -> 'a m -> 'a m
val null : 'a m -> bool
null x implies that x is zero. If you do not want to or cannot answer whether a given x is zero, then null x should be false. I have provided this so that streams can be implemented more efficiently.
